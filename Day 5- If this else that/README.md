## Things I learnt today

### If statements:
```python
name = "John"

if name == "John":
  print("Hello John!")
else:
  print("Hi")

# **Case 1**
# Input: John
# Output: Hello John!
# 
# **Case 2**
# Input: David
# Output: Hi
```


These statements are used to say the computer to do something depending on a condition we specify.
Either the code in the if statement or else statement gets executed, not both.
